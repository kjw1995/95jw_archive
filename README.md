## 기술 스택

- **Static Site Generator**: Hugo
- **Theme**: Hextra
- **Hosting**: GitHub Pages
- **CI/CD**: GitHub Actions

## 로컬 개발

```bash
# 클론
git clone https://github.com/kjw1995/95jw_archive.git
cd 95jw_archive

# 서버 실행
hugo mod tidy
hugo server --logLevel debug --disableFastRender -p 1313
```
