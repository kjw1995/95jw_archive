## 1. 가정 네트워크 전체 구조

가정 네트워크는 ISP의 WAN을 내 집 안의 LAN에 연결한 작은 시스템이다. 핵심 장비는 모뎀과 공유기(라우터)다.

```
     인터넷 (WAN)
         │
      ┌──┴──┐
      │모뎀 │  ISP 연결
      └──┬──┘
         │
      ┌──┴──┐
      │공유기│  LAN 생성
      └──┬──┘
    ┌────┼────┐
  ┌─┴┐ ┌─┴┐ ┌─┴┐
  │PC│ │폰│ │TV│
  └──┘ └──┘ └──┘
```

---

## 2. 인터넷 서비스 제공자 (ISP)

ISP는 가정이나 기업에 인터넷 접속 서비스를 제공하는 사업자다.

### 국내 주요 ISP

| ISP | 특징 |
|:---|:---|
| KT | 국내 최대 인프라 |
| SK브로드밴드 | SK 계열, 결합상품 강점 |
| LG U+ | TV 결합 상품 강점 |
| 지역 케이블사 | 지역 기반 서비스 |

### ISP가 제공하는 것

1. 공인 IP 주소 할당
2. 물리 회선 (광케이블 등)
3. DNS 서버
4. 이메일·웹호스팅 등 부가 서비스

---

## 3. 인터넷 회선의 종류

| 회선 | 매체 | 속도 | 특징 |
|:---|:---|:---|:---|
| FTTH (광) | 광케이블 | 100Mbps~10Gbps | 고속·안정, 설치비 높음 |
| VDSL | 전화선 | ~300Mbps | 기존 인프라 활용 |
| ADSL | 전화선 | ~50Mbps | 구형 |
| 케이블 | 동축 | 10Mbps~1Gbps | 이웃과 공유 |

---

## 4. 공유기 (Broadband Router)

가정용 공유기 한 대가 라우터·스위치·AP·DHCP·NAT·방화벽 기능을 통합 제공한다.

### 주요 기능

| 기능 | 설명 |
|:---|:---|
| 라우팅 | WAN↔LAN 패킷 전달 |
| NAT | 공인 IP 1개로 여러 기기 인터넷 |
| DHCP | 내부 기기에 자동 IP 할당 |
| 스위칭 허브 | 유선 포트 간 프레임 전달 |
| 무선 AP | Wi-Fi 송출 |
| 방화벽 | 외부 침입 기본 차단 |

### NAT 개념

```
사설 IP              공인 IP
┌──────────┐        ┌──────────┐
│192.168.  │        │          │
│  0.2     │        │          │
│  0.3     │ ──NAT─▶│211.x.x.x │──▶인터넷
│  0.4     │        │          │
└──────────┘        └──────────┘
```

### DHCP 절차

```
기기                   공유기
 │  DHCP Discover       │
 ├──────────────────────▶
 │  DHCP Offer          │
 ◀──────────────────────┤
 │  DHCP Request        │
 ├──────────────────────▶
 │  DHCP Ack            │
 ◀──────────────────────┤
```

## 5. 공인 IP와 사설 IP

| 구분 | 공인 IP | 사설 IP |
|:---|:---|:---|
| 할당 | ISP | 공유기 (DHCP) |
| 범위 | 인터넷 전역 고유 | 내부망 한정 |
| 외부 접근 | 가능 | 불가 (NAT 필요) |

### 사설 IP 대역 (RFC 1918)

| 클래스 | 대역 |
|:---|:---|
| A | 10.0.0.0/8 |
| B | 172.16.0.0/12 |
| C | 192.168.0.0/16 |

{{< callout type="info" >}}
대부분의 가정 공유기는 기본 게이트웨이로 `192.168.0.1` 또는 `192.168.1.1`을 사용한다. 관리자 페이지 접속 주소이기도 하다.
{{< /callout >}}

## 6. 유선 LAN vs 무선 LAN

| 항목 | 유선 LAN | 무선 LAN |
|:---|:---|:---|
| 매체 | UTP 케이블 | 전파 (2.4/5/6GHz) |
| 속도 | 1~10 Gbps | ~9.6 Gbps (Wi-Fi 6) |
| 안정성 | 높음 | 간섭·거리 영향 |
| 이동성 | 제약 | 자유 |
| 용도 | 데스크탑·서버 | 모바일·노트북 |

### Wi-Fi 세대

| 세대 | 규격 | 최대 속도 | 주파수 |
|:---|:---|:---|:---|
| Wi-Fi 4 | 802.11n | 600Mbps | 2.4/5GHz |
| Wi-Fi 5 | 802.11ac | 3.5Gbps | 5GHz |
| Wi-Fi 6 | 802.11ax | 9.6Gbps | 2.4/5GHz |
| Wi-Fi 6E | 802.11ax | 9.6Gbps | 2.4/5/6GHz |
| Wi-Fi 7 | 802.11be | 46Gbps | 2.4/5/6GHz |

## 7. IP 확인 명령어

### Windows

```bash
ipconfig
```

결과 예시:

```
이더넷 어댑터 이더넷:
  IPv4 주소 . . . . . . : 192.168.0.10
  서브넷 마스크 . . . . : 255.255.255.0
  기본 게이트웨이 . . . : 192.168.0.1
```

### Linux / macOS

```bash
ip addr         # Linux
ifconfig        # macOS·구형 Linux
```

## 8. 보안 기본

### 권장 조치

1. 공유기 관리자 비밀번호 변경
2. Wi-Fi 암호화는 WPA2-PSK 이상 (가능하면 WPA3)
3. SSID 기본값 변경
4. 펌웨어 주기적 업데이트

### 무선 보안 프로토콜

| 프로토콜 | 보안 수준 | 권장 여부 |
|:---|:---|:---|
| WEP | 매우 낮음 | 사용 금지 |
| WPA | 낮음 | 비권장 |
| WPA2-PSK | 양호 | 권장 |
| WPA3 | 높음 | 강력 권장 |

## 핵심 정리

| 구성 요소 | 역할 |
|:---|:---|
| ISP | 인터넷 접속 서비스 제공 |
| 모뎀 | ISP 회선과 연결 |
| 공유기 | NAT·DHCP·라우팅·방화벽 |
| 사설 IP | 내부망 식별 (192.168.x.x) |
| 공인 IP | 인터넷 전역 식별 |

{{< callout type="info" >}}
**인터넷 공유기 (Broadband Router)**
가정이나 소규모 사무실용 통합 장비. 라우터, 스위칭 허브, 무선 AP, DHCP, NAT, 방화벽을 한 장비에 결합해 제공한다.
{{< /callout >}}
