## 1. 전송 매체 개요

### 전송 매체란?

데이터가 흐르는 물리적인 선로를 전송 매체라고 한다. 전기 신호나 빛 신호가 흐르는 통로 역할을 하며, 유선 매체와 무선 매체로 구분한다.

```
┌──────────────────────────────┐
│          전송 매체           │
├──────────────┬───────────────┤
│     유선     │     무선      │
├──────────────┼───────────────┤
│ 트위스트페어 │  라디오파     │
│ 동축 케이블  │  마이크로파   │
│ 광케이블     │  적외선       │
└──────────────┴───────────────┘
```

### 유선 매체 비교

| 종류 | 전송 매체 | 특징 |
|:---|:---|:---|
| 트위스트 페어 | 구리선 | 저렴, 노이즈 영향 |
| 동축 케이블 | 구리선 | 중거리, 레거시 |
| 광케이블 | 유리/플라스틱 | 장거리 고속 |

## 2. 트위스트 페어 케이블

### 구조와 원리

구리선 8개를 2개씩 꼬아 4쌍으로 만든 케이블이다. 꼬는 이유는 선 주변에 발생하는 자기장을 서로 상쇄시켜 전자기 간섭(EMI)을 줄이기 위해서다.

```
    노이즈 ↓↓↓↓↓
   ╱╲   ╱╲   ╱╲   ╱╲
  ╱  ╲ ╱  ╲ ╱  ╲ ╱  ╲
 ↑  ↓  ↑  ↓  ↑  ↓
 노이즈가 서로 상쇄
```

{{< callout type="info" >}}
두 선을 꼬면 각 구간에서 유도되는 노이즈 극성이 반대가 되어 수신측에서 차분 신호로 복원할 때 상쇄된다. 차동 신호 전송(differential signaling)의 기본 원리이다.
{{< /callout >}}

### UTP와 STP

| 항목 | UTP | STP |
|:---|:---|:---|
| 차폐 | 없음 | 금속 실드 |
| 노이즈 저항 | 보통 | 우수 |
| 가격 | 저렴 | 비쌈 |
| 유연성 | 좋음 | 떨어짐 |
| 접지 | 불필요 | 필요 |
| 용도 | 사무실·가정 | 공장·의료 |

## 3. 카테고리별 분류

### 카테고리(CAT) 규격

| 카테고리 | 규격 | 최대 속도 | 대역폭 |
|:---|:---|:---|:---|
| Cat 5 | 100BASE-TX | 100 Mbps | 100 MHz |
| Cat 5e | 1000BASE-T | 1 Gbps | 100 MHz |
| Cat 6 | 1000BASE-TX | 1 Gbps | 250 MHz |
| Cat 6a | 10GBASE-T | 10 Gbps | 500 MHz |
| Cat 7 | 10GBASE-T | 10 Gbps | 600 MHz |
| Cat 8 | 40GBASE-T | 40 Gbps | 2000 MHz |

최대 거리는 Cat 8(30m)을 제외하면 대부분 100m이다.

{{< callout type="info" >}}
현재 신규 설치 시 가정·사무실은 Cat 6, 10G가 필요한 서버룸은 Cat 6a 이상을 권장한다. Cat 5는 단종되어 사용하지 않는다.
{{< /callout >}}

## 4. RJ-45 커넥터

### 핀 배열과 배선 표준

트위스트 페어 케이블에는 8핀(8P8C) 구조의 RJ-45 커넥터를 사용한다. 배선 방식은 T568A와 T568B 두 표준이 있으며, 현재는 T568B를 주로 사용한다.

| 핀 | T568A | T568B |
|:---|:---|:---|
| 1 | 흰색/녹색 | 흰색/주황 |
| 2 | 녹색 | 주황 |
| 3 | 흰색/주황 | 흰색/녹색 |
| 4 | 파랑 | 파랑 |
| 5 | 흰색/파랑 | 흰색/파랑 |
| 6 | 주황 | 녹색 |
| 7 | 흰색/갈색 | 흰색/갈색 |
| 8 | 갈색 | 갈색 |

### 다이렉트와 크로스 케이블

```
다이렉트: 양쪽 모두 T568B
[PC] ──── [스위치]

크로스: 한쪽 A, 다른쪽 B
[PC] ──── [PC]
```

| 연결 | 케이블 |
|:---|:---|
| PC ↔ 스위치 | 다이렉트 |
| 스위치 ↔ 라우터 | 다이렉트 |
| PC ↔ PC | 크로스 |
| 스위치 ↔ 스위치 | 크로스 |

{{< callout type="info" >}}
기가비트 이상 장비는 Auto MDI-X 기능을 내장하여 송수신 핀을 자동 전환한다. 따라서 실무에서는 다이렉트 케이블만 사용해도 문제없이 동작한다.
{{< /callout >}}

## 5. 광케이블

### 구조

```
  ┌─────────────────────┐
  │   외피(Jacket)      │
  │  ┌───────────────┐  │
  │  │ 클래딩         │  │
  │  │  ┌─────────┐  │  │
  │  │  │  코어   │  │  │
  │  │  │  → 빛 → │  │  │
  │  │  └─────────┘  │  │
  │  └───────────────┘  │
  └─────────────────────┘
```

코어는 빛이 지나가는 중심부이고, 클래딩은 코어보다 굴절률이 낮아 빛을 전반사로 가두는 층이다.

### 싱글모드와 멀티모드

| 항목 | 싱글모드(SMF) | 멀티모드(MMF) |
|:---|:---|:---|
| 코어 직경 | 9μm | 50/62.5μm |
| 전송 거리 | 수십 km | 수백 m~2 km |
| 광원 | 레이저 | LED/VCSEL |
| 비용 | 높음 | 낮음 |
| 용도 | 장거리·백본 | 건물 내부 |

## 6. 동축 케이블

중심 도체와 외부 도체가 동일한 축을 공유하는 케이블이다. 과거 이더넷(10BASE5/10BASE2)에 사용되었고, 현재는 케이블 TV·CCTV 등에 주로 쓰인다.

| 종류 | 임피던스 | 용도 |
|:---|:---|:---|
| RG-58 | 50Ω | 구형 이더넷 |
| RG-59 | 75Ω | CCTV |
| RG-6 | 75Ω | 케이블 TV·위성 |

## 핵심 정리

| 개념 | 설명 |
|:---|:---|
| 트위스트 페어 | 구리선 4쌍을 꼬아 EMI를 줄인 케이블 |
| UTP / STP | 차폐 없음 / 금속 실드 |
| 카테고리 | 케이블 성능 등급(Cat 5e~Cat 8) |
| RJ-45 | 8핀 모듈러 커넥터 |
| 광케이블 | 빛으로 데이터를 전송, 장거리·고속 |

{{< callout type="info" >}}
**용어 정리**
- **UTP / STP**: 비차폐 / 차폐 연선
- **RJ-45**: 8핀 8접점(8P8C) 커넥터
- **T568A / T568B**: 배선 색상 표준
- **Auto MDI-X**: 송수신 핀을 자동 전환하는 기능
- **SMF / MMF**: 싱글모드 / 멀티모드 광케이블
{{< /callout >}}
