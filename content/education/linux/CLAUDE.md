# 스타일 컨벤션 (06·07 기준)
- 테이블: 텍스트 컬럼 좌측 정렬 `|:-----|`, 단일 기호/키 컬럼만 가운데 `|:--:|`
- 다이어그램: 코드 블록 다이어그램 규칙 준수 (폭 40자 이내, 한글 2칸)
  - 단순 나열은 박스로 감싸지 말고 리스트/표/callout로
- 주의·팁은 Hextra callout 사용 `{{</* callout type="warning|info" */>}}`
