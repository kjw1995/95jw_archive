## 핵심 키워드

- **동시성 vs 병렬성**: 논리적 동시 처리 vs 물리적 동시 처리
- **스레드 안전성**: 여러 스레드가 접근해도 올바르게 동작
- **경쟁 조건**: 여러 스레드가 공유 자원에 동시 접근
- **교착 상태**: 스레드들이 서로의 자원을 기다리며 무한 대기
- **가시성**: 한 스레드의 변경이 다른 스레드에게 보이는 것
