# 이더넷의 종류와 특징

## 1. 이더넷 규격 개요

### 이더넷 규격이란?
- 이더넷은 **케이블 종류**와 **통신 속도**에 따라 다양한 규격으로 분류
- IEEE 802.3 표준으로 정의
- 규격 이름에 속도, 전송 방식, 케이블 정보 포함

### 이더넷 규격 명명법

```
┌─────────────────────────────────────────────────────────────┐
│                   이더넷 규격 명명법                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│         10   BASE  -  T                                     │
│         │     │       │                                     │
│         │     │       └── 케이블 종류 또는 특성              │
│         │     │           T: Twisted Pair (UTP)             │
│         │     │           F: Fiber (광케이블)                │
│         │     │           S: Short wavelength               │
│         │     │           L: Long wavelength                │
│         │     │           숫자: 케이블 길이 (×100m)          │
│         │     │                                             │
│         │     └── 전송 방식                                  │
│         │         BASE: Baseband (디지털, 단일 채널)         │
│         │         BROAD: Broadband (아날로그, 다중 채널)     │
│         │                                                   │
│         └── 통신 속도 (Mbps)                                 │
│             10 = 10Mbps                                     │
│             100 = 100Mbps                                   │
│             1000 = 1Gbps                                    │
│             10G = 10Gbps                                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 명명법 예시

```
10BASE5:
─────────────────────────────────────────
10    = 10Mbps
BASE  = Baseband 전송
5     = 최대 500m (×100)

10BASE-T:
─────────────────────────────────────────
10    = 10Mbps
BASE  = Baseband 전송
T     = Twisted Pair (UTP 케이블)

1000BASE-SX:
─────────────────────────────────────────
1000  = 1000Mbps (1Gbps)
BASE  = Baseband 전송
S     = Short wavelength (단파장)
X     = 광케이블 (Fiber)
```

## 2. 주요 이더넷 규격

### 규격 요약표

| 규격 이름 | 통신 속도 | 케이블 | 최대 길이 | 표준화 연도 |

## 3. 초기 이더넷 (10Mbps)

### 10BASE5 (Thick Ethernet)

```
┌─────────────────────────────────────────────────────────────┐
│                        10BASE5                               │
│                    "Thick Ethernet"                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  속도: 10Mbps                                               │
│  케이블: 두꺼운 동축 케이블 (직경 약 10mm)                    │
│  최대 길이: 500m (세그먼트당)                                │
│  표준화: 1982년 (IEEE 802.3)                                │
│                                                             │
│  구조:                                                      │
│  ════════════════════════════════════════════               │
│       │          │          │          │                    │
│     [AUI]      [AUI]      [AUI]      [AUI]                 │
│       │          │          │          │                    │
│    [트랜시버] [트랜시버] [트랜시버] [트랜시버]               │
│       │          │          │          │                    │
│     [PC]       [PC]       [PC]       [PC]                  │
│                                                             │
│  특징:                                                      │
│  • 최초의 이더넷 표준                                        │
│  • 트랜시버로 케이블에 연결                                  │
│  • 버스 토폴로지                                            │
│  • 현재 사용하지 않음                                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 10BASE2 (Thin Ethernet)

```
┌─────────────────────────────────────────────────────────────┐
│                        10BASE2                               │
│                    "Thin Ethernet"                           │
│                    "Cheapernet"                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  속도: 10Mbps                                               │
│  케이블: 얇은 동축 케이블 (RG-58, 직경 약 5mm)               │
│  최대 길이: 185m (세그먼트당)                                │
│  표준화: 1985년 (IEEE 802.3a)                               │
│                                                             │
│  구조:                                                      │
│  ─────[T]─────[T]─────[T]─────[T]─────                     │
│        │       │       │       │                            │
│      [PC]    [PC]    [PC]    [PC]                          │
│                                                             │
│  [T] = T커넥터 (BNC)                                        │
│                                                             │
│  특징:                                                      │
│  • 10BASE5보다 저렴하고 유연                                 │
│  • BNC 커넥터 사용                                          │
│  • 버스 토폴로지                                            │
│  • 양 끝에 터미네이터 필요                                   │
│  • 현재 사용하지 않음                                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 10BASE-T

```
┌─────────────────────────────────────────────────────────────┐
│                       10BASE-T                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  속도: 10Mbps                                               │
│  케이블: UTP 케이블 (Cat3 이상)                              │
│  커넥터: RJ-45                                              │
│  최대 길이: 100m                                            │
│  표준화: 1990년 (IEEE 802.3i)                               │
│                                                             │
│  구조 (스타 토폴로지):                                       │
│                                                             │
│          [PC]     [PC]                                      │
│            │       │                                        │
│            └───┬───┘                                        │
│                │                                            │
│          ┌─────┴─────┐                                      │
│          │  HUB or   │                                      │
│          │  SWITCH   │                                      │
│          └─────┬─────┘                                      │
│                │                                            │
│            ┌───┴───┐                                        │
│            │       │                                        │
│          [PC]    [PC]                                       │
│                                                             │
│  특징:                                                      │
│  • UTP 케이블 사용 시작 (혁신!)                              │
│  • 스타 토폴로지로 변경                                      │
│  • 설치 및 관리 용이                                         │
│  • 허브/스위치 사용                                          │
│  • 현재 레거시 시스템에서 일부 사용                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 4. Fast Ethernet (100Mbps)

### 100BASE-TX

```
┌─────────────────────────────────────────────────────────────┐
│                      100BASE-TX                              │
│                    "Fast Ethernet"                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  속도: 100Mbps                                              │
│  케이블: UTP 케이블 (Cat5 이상)                              │
│  커넥터: RJ-45                                              │
│  최대 길이: 100m                                            │
│  표준화: 1995년 (IEEE 802.3u)                               │
│                                                             │
│  케이블 사용:                                               │
│  ┌─────────────────────────────────────────┐               │
│  │  쌍 1 (1,2번): 송신 (TX)                 │               │
│  │  쌍 2 (3,6번): 수신 (RX)                 │               │
│  │  쌍 3 (4,5번): 미사용                    │               │
│  │  쌍 4 (7,8번): 미사용                    │               │
│  └─────────────────────────────────────────┘               │
│                                                             │
│  특징:                                                      │
│  • 10BASE-T의 10배 속도                                     │
│  • 같은 UTP 케이블 인프라 사용 가능                          │
│  • 전이중/반이중 모두 지원                                   │
│  • 가장 널리 보급된 이더넷 규격 중 하나                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 100BASE-FX

```
┌─────────────────────────────────────────────────────────────┐
│                      100BASE-FX                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  속도: 100Mbps                                              │
│  케이블: 광케이블 (멀티모드, MMF)                            │
│  커넥터: SC, ST, LC                                         │
│  최대 길이: 2km (전이중)                                    │
│  표준화: 1995년 (IEEE 802.3u)                               │
│                                                             │
│  특징:                                                      │
│  • 장거리 전송 가능                                          │
│  • 전자기 간섭 없음                                          │
│  • 건물 간 연결에 사용                                       │
│  • 백본 네트워크에 적합                                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 5. Gigabit Ethernet (1Gbps)

### 1000BASE-T

```
┌─────────────────────────────────────────────────────────────┐
│                      1000BASE-T                              │
│                   "Gigabit Ethernet"                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  속도: 1000Mbps (1Gbps)                                     │
│  케이블: UTP 케이블 (Cat5e 이상, Cat6 권장)                  │
│  커넥터: RJ-45                                              │
│  최대 길이: 100m                                            │
│  표준화: 1999년 (IEEE 802.3ab)                              │
│                                                             │
│  케이블 사용 (4쌍 모두 사용):                                │
│  ┌─────────────────────────────────────────┐               │
│  │  쌍 1 (1,2번): 양방향 250Mbps            │               │
│  │  쌍 2 (3,6번): 양방향 250Mbps            │               │
│  │  쌍 3 (4,5번): 양방향 250Mbps            │               │
│  │  쌍 4 (7,8번): 양방향 250Mbps            │               │
│  │  ─────────────────────────────          │               │
│  │  총합: 250 × 4 = 1000Mbps               │               │
│  └─────────────────────────────────────────┘               │
│                                                             │
│  특징:                                                      │
│  • 4쌍 모두 사용 (하이브리드 전송)                           │
│  • 각 쌍이 양방향 통신 (에코 캔슬레이션)                     │
│  • 기존 Cat5e 케이블 활용 가능                               │
│  • 현재 가장 보편적인 LAN 규격                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 6. 10 Gigabit Ethernet (10Gbps)

## 8. 이더넷 발전 역사

### 연대별 발전

```
이더넷 발전 타임라인:

1973년 ──── Xerox PARC에서 최초 개발 (2.94Mbps)
   │
1980년 ──── DIX 이더넷 발표 (DEC, Intel, Xerox)
   │
1982년 ──── 10BASE5 (IEEE 802.3)
   │        └── 최초 표준, 동축 케이블
   │
1985년 ──── 10BASE2
   │        └── 얇은 동축 케이블, 저렴
   │
1990년 ──── 10BASE-T ★
   │        └── UTP 케이블 시대 시작!
   │
1995년 ──── 100BASE-TX (Fast Ethernet) ★
   │        └── 100Mbps, 대중화
   │
1998년 ──── 1000BASE-X (Gigabit, 광케이블)
   │
1999년 ──── 1000BASE-T (Gigabit, UTP) ★
   │        └── 현재 LAN 표준
   │
2002년 ──── 10GBASE-X (10G, 광케이블)
   │
2006년 ──── 10GBASE-T (10G, UTP)
   │        └── 데이터센터 표준
   │
2010년 ──── 40G/100G Ethernet
   │
2017년 ──── 200G/400G Ethernet
   │
현재  ──── 800G/1.6T Ethernet 개발 중

★ = 중요 전환점
```

## 9. 케이블 카테고리와 규격 매칭

### UTP 케이블 카테고리

| 카테고리 | 대역폭 | 지원 규격 | 비고 |
|----------|--------|-----------|------|
| Cat3 | 16MHz | 10BASE-T | 전화선, 구형 |
| Cat5 | 100MHz | 100BASE-TX | 단종, 비권장 |
| Cat5e | 100MHz | 1000BASE-T | 현재 최소 권장 |

## 핵심 정리

| 개념 | 설명 |
|------|------|
| 이더넷 규격 | 속도, 전송 방식, 케이블 종류에 따른 표준 |
| BASE | Baseband 전송 방식 (디지털, 단일 채널) |
| 10BASE-T | 10Mbps, UTP 케이블, 스타 토폴로지 시작 |
| 100BASE-TX | 100Mbps, Fast Ethernet |
| 1000BASE-T | 1Gbps, 현재 LAN 표준 |
| 10GBASE-T | 10Gbps, 데이터센터 표준 |

### 규격 명명법 요약

```
┌─────────────────────────────────────────────────────────────┐
│                    규격 명명법 요약                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│     [속도] [전송방식] - [케이블/특성]                        │
│                                                             │
│  속도:                                                      │
│  • 10, 100, 1000: Mbps 단위                                │
│  • 10G, 40G, 100G: Gbps 단위                               │
│                                                             │
│  전송방식:                                                  │
│  • BASE: Baseband (일반적)                                 │
│                                                             │
│  케이블/특성:                                               │
│  • T: Twisted Pair (UTP)                                   │
│  • F, FX: Fiber (광케이블)                                 │
│  • S, SX, SR: Short (단거리/단파장)                        │
│  • L, LX, LR: Long (장거리/장파장)                         │
│  • 숫자: 케이블 길이 ×100m (동축)                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

{{< callout type="info" >}}
**용어 정리**
- **Baseband**: 디지털 신호를 그대로 전송하는 방식, 단일 채널 사용
- **UTP (Unshielded Twisted Pair)**: 비차폐 꼬임 쌍선 케이블
- **MMF (Multi-Mode Fiber)**: 멀티모드 광케이블, 단거리용
- **SMF (Single-Mode Fiber)**: 싱글모드 광케이블, 장거리용
- **Fast Ethernet**: 100Mbps 이더넷 (100BASE-TX)
- **Gigabit Ethernet**: 1Gbps 이더넷 (1000BASE-T)
- **10G Ethernet**: 10Gbps 이더넷 (10GBASE-T)
{{< /callout >}}
