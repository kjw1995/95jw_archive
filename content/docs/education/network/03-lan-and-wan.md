## 네트워크의 범위에 따른 분류

> 네트워크는 **접속 범위**에 따라 분류

```
┌─────────────────────────────────────────────────────────────────────┐
│                    네트워크 범위에 따른 분류                          │
│                                                                     │
│   ┌─────────┐                                                      │
│   │   PAN   │  Personal Area Network (개인 영역)                   │
│   │  ~10m   │  블루투스, USB 연결                                   │
│   └────┬────┘                                                      │
│        │                                                            │
│   ┌────▼────┐                                                      │
│   │   LAN   │  Local Area Network (근거리)                         │
│   │ ~수 km  │  건물, 사무실, 학교                                   │
│   └────┬────┘                                                      │
│        │                                                            │
│   ┌────▼────┐                                                      │
│   │   MAN   │  Metropolitan Area Network (도시)                    │
│   │ ~수십km │  도시 단위 네트워크                                   │
│   └────┬────┘                                                      │
│        │                                                            │
│   ┌────▼────┐                                                      │
│   │   WAN   │  Wide Area Network (광역)                            │
│   │  전세계  │  국가/대륙 간 연결                                   │
│   └─────────┘                                                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 1. LAN (Local Area Network)

> **근거리 통신망** - 건물 안이나 특정 지역을 범위로 하는 네트워크

```
┌─────────────────────────────────────────────────────────────────────┐
│                         LAN 구성 예시                                │
│                                                                     │
│   ┌───────────────────────── 사무실 ─────────────────────────┐     │
│   │                                                           │     │
│   │    ┌────┐   ┌────┐   ┌────┐   ┌────┐                    │     │
│   │    │ PC │   │ PC │   │ PC │   │ PC │                    │     │
│   │    └──┬─┘   └──┬─┘   └──┬─┘   └──┬─┘                    │     │
│   │       │        │        │        │                       │     │
│   │       └────────┴────┬───┴────────┘                       │     │
│   │                     │                                    │     │
│   │              ┌──────┴──────┐                             │     │
│   │              │   스위치    │                              │     │
│   │              └──────┬──────┘                             │     │
│   │                     │                                    │     │
│   │    ┌────────────────┼────────────────┐                   │     │
│   │    │                │                │                   │     │
│   │  ┌─┴──┐         ┌───┴───┐        ┌──┴───┐              │     │
│   │  │서버│         │프린터 │        │ NAS  │              │     │
│   │  └────┘         └───────┘        └──────┘              │     │
│   │                                                          │     │
│   └──────────────────────────────────────────────────────────┘     │
│                                                                     │
│   특징: 고속, 자체 구축/관리, 제한된 범위                            │
└─────────────────────────────────────────────────────────────────────┘
```

### LAN의 특징

| 특징 | 설명 |
|------|------|
| **범위** | 건물, 층, 캠퍼스 (수 km 이내) |
| **속도** | 고속 (100Mbps ~ 10Gbps) |
| **소유** | 조직이 직접 구축/관리 |
| **비용** | 초기 구축 비용, 유지보수 직접 |
| **오류율** | 낮음 (환경 통제 가능) |
| **지연** | 매우 낮음 (수 ms 이하) |

### LAN 구성 장비

```
┌─────────────────────────────────────────────────────────────────────┐
│                       LAN 핵심 장비                                  │
│                                                                     │
│   ┌─────────────┐    ┌─────────────┐    ┌─────────────┐            │
│   │   스위치     │    │   라우터    │    │   공유기    │            │
│   │  (Switch)   │    │  (Router)   │    │ (AP Router) │            │
│   ├─────────────┤    ├─────────────┤    ├─────────────┤            │
│   │ L2 (MAC)    │    │ L3 (IP)     │    │ 스위치+라우터 │            │
│   │ 같은 네트워크│    │ 다른 네트워크│    │ +무선AP 결합 │            │
│   │ 내부 연결    │    │ 간 연결     │    │ 가정용 통합  │            │
│   └─────────────┘    └─────────────┘    └─────────────┘            │
│                                                                     │
│   ┌─────────────┐    ┌─────────────┐    ┌─────────────┐            │
│   │   NIC       │    │   케이블    │    │   허브      │            │
│   │ (랜카드)    │    │  (UTP/광)   │    │  (Hub)     │            │
│   ├─────────────┤    ├─────────────┤    ├─────────────┤            │
│   │ PC-네트워크 │    │ 장비 간     │    │ 단순 신호   │            │
│   │ 연결 인터페 │    │ 물리적 연결 │    │ 복제 (구식) │            │
│   │ 이스        │    │             │    │             │            │
│   └─────────────┘    └─────────────┘    └─────────────┘            │
└─────────────────────────────────────────────────────────────────────┘
```

## 2. WAN (Wide Area Network)

> **광역 통신망** - ISP가 제공하는 서비스로 구축된 광범위 네트워크

```
┌─────────────────────────────────────────────────────────────────────┐
│                         WAN 구성 개념                                │
│                                                                     │
│      서울 지사                              부산 본사               │
│   ┌────────────┐                         ┌────────────┐            │
│   │    LAN     │                         │    LAN     │            │
│   │  ┌──┐┌──┐  │                         │  ┌──┐┌──┐  │            │
│   │  │PC││PC│  │                         │  │PC││PC│  │            │
│   │  └──┘└──┘  │                         │  └──┘└──┘  │            │
│   │     │      │                         │     │      │            │
│   │  ┌──┴──┐   │                         │  ┌──┴──┐   │            │
│   │  │라우터│   │                         │  │라우터│   │            │
│   └──┴──┬──┴───┘                         └──┴──┬──┴───┘            │
│         │                                      │                   │
│         │         ┌─────────────────┐          │                   │
│         └─────────┤                 ├──────────┘                   │
│                   │   ISP 네트워크  │                              │
│                   │   (인터넷)      │                              │
│                   │  KT, SKT, LG   │                              │
│                   └─────────────────┘                              │
│                                                                     │
│   특징: LAN들을 연결, ISP 서비스 이용, 넓은 범위                     │
└─────────────────────────────────────────────────────────────────────┘
```

### WAN의 특징

| 특징 | 설명 |
|------|------|
| **범위** | 도시, 국가, 대륙, 전세계 |
| **속도** | LAN보다 느림 (수 Mbps ~ 수 Gbps) |
| **소유** | ISP가 소유/관리 (임대 사용) |
| **비용** | 월정액 또는 사용량 기반 요금 |
| **오류율** | 상대적으로 높음 |
| **지연** | 상대적으로 높음 (수십~수백 ms) |

### ISP (Internet Service Provider)

> **인터넷 서비스 제공자** - 인터넷 접속 서비스를 제공하는 사업자

```
┌─────────────────────────────────────────────────────────────────────┐
│                      ISP 계층 구조                                   │
│                                                                     │
│                    ┌─────────────────┐                              │
│                    │   Tier 1 ISP    │  국제 백본망                 │
│                    │ (글로벌 사업자)  │  AT&T, NTT, Level3          │
│                    └────────┬────────┘                              │
│                             │                                       │
│              ┌──────────────┼──────────────┐                       │
│              │              │              │                       │
│       ┌──────┴─────┐ ┌──────┴─────┐ ┌──────┴─────┐                 │
│       │ Tier 2 ISP │ │ Tier 2 ISP │ │ Tier 2 ISP │  국가/지역      │
│       │  (KT)      │ │  (SKT)     │ │  (LGU+)    │  사업자         │
│       └──────┬─────┘ └──────┬─────┘ └──────┬─────┘                 │
│              │              │              │                       │
│       ┌──────┴─────┐ ┌──────┴─────┐ ┌──────┴─────┐                 │
│       │ Tier 3 ISP │ │ Tier 3 ISP │ │ Tier 3 ISP │  지역/소규모    │
│       │ (지역업체)  │ │ (케이블TV) │ │ (알뜰폰)   │  사업자         │
│       └──────┬─────┘ └────────────┘ └────────────┘                 │
│              │                                                      │
│       ┌──────┴──────┐                                              │
│       │   최종 사용자 │  가정, 기업, 학교                           │
│       └─────────────┘                                              │
└─────────────────────────────────────────────────────────────────────┘
```

### 한국의 주요 ISP

| ISP | 서비스명 | 특징 |
|-----|---------|------|
| **KT** | olleh | 가장 넓은 인프라 |
| **SK브로드밴드** | B tv | SK 계열 |
| **LG U+** | U+ 인터넷 | LG 계열 |
| **SK텔레콤** | T 인터넷 | 모바일 연계 |

## 3. LAN vs WAN 비교

```
┌─────────────────────────────────────────────────────────────────────┐
│                       LAN vs WAN 비교                                │
│                                                                     │
│              LAN                           WAN                      │
│   ┌─────────────────────┐       ┌─────────────────────┐            │
│   │                     │       │                     │            │
│   │    ████████████     │       │    ▓▓▓▓▓▓▓▓▓▓▓▓    │            │
│   │    █ 고속 전송 █     │       │    ▓ 저속 전송  ▓    │            │
│   │    ████████████     │       │    ▓▓▓▓▓▓▓▓▓▓▓▓    │            │
│   │                     │       │                     │            │
│   │   건물/캠퍼스 내     │       │   도시/국가/세계     │            │
│   │   직접 구축/관리     │       │   ISP 서비스 이용   │            │
│   │   오류 적음         │       │   오류 상대적 많음   │            │
│   │                     │       │                     │            │
│   └─────────────────────┘       └─────────────────────┘            │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 상세 비교표

| 비교 항목 | LAN | WAN |
|----------|-----|-----|
| **전체 이름** | Local Area Network | Wide Area Network |
| **한국어** | 근거리 통신망 | 광역 통신망 |
| **범위** | 좁음 (건물, 캠퍼스) | 넓음 (도시, 국가, 세계) |
| **속도** | 빠름 (100Mbps~10Gbps) | 느림 (수 Mbps~Gbps) |
| **지연시간** | 낮음 (<1ms) | 높음 (10~300ms) |
| **오류율** | 낮음 | 높음 |
| **구축비용** | 초기 투자 | 월 이용료 |
| **소유/관리** | 조직 자체 | ISP |
| **혼잡도** | 통제 가능 | 예측 어려움 |
| **보안** | 통제 용이 | 복잡함 |

## 4. MAN (Metropolitan Area Network)

> **도시권 통신망** - LAN과 WAN 사이의 규모

```
┌─────────────────────────────────────────────────────────────────────┐
│                         MAN 개념                                     │
│                                                                     │
│   ┌─ 도시 전체 (서울시) ───────────────────────────────────────┐    │
│   │                                                            │    │
│   │   강남구           서초구           송파구                  │    │
│   │  ┌─────┐         ┌─────┐         ┌─────┐                 │    │
│   │  │ LAN │─────────│ LAN │─────────│ LAN │                 │    │
│   │  └──┬──┘         └──┬──┘         └──┬──┘                 │    │
│   │     │               │               │                     │    │
│   │     └───────────────┼───────────────┘                     │    │
│   │                     │                                     │    │
│   │              ┌──────┴──────┐                              │    │
│   │              │  MAN 백본   │                              │    │
│   │              │  (광케이블) │                              │    │
│   │              └─────────────┘                              │    │
│   │                                                            │    │
│   └────────────────────────────────────────────────────────────┘    │
│                                                                     │
│   예시: 지하철 와이파이, 도시 CCTV망, 케이블TV 네트워크              │
└─────────────────────────────────────────────────────────────────────┘
```

| 특징 | 설명 |
|------|------|
| **범위** | 도시 단위 (수십 km) |
| **기술** | 주로 광케이블 사용 |
| **용도** | 도시 인프라, 대학 캠퍼스 연결 |
| **예시** | 지하철망, 케이블TV, 대규모 캠퍼스 |

## 5. 인터넷의 역사

> WAN의 시초 - ARPANET

```
┌─────────────────────────────────────────────────────────────────────┐
│                       인터넷 역사 타임라인                           │
│                                                                     │
│   1969년 ─────────────────────────────────────────────────────      │
│   │  ARPANET 시작 (컴퓨터 4대 연결)                                 │
│   │  - UCLA, 스탠포드 연구소, UCSB, 유타대학                        │
│   │                                                                 │
│   1970년대 ───────────────────────────────────────────────────      │
│   │  대학/연구소 네트워크 확산                                       │
│   │  - TCP/IP 프로토콜 개발                                         │
│   │                                                                 │
│   1980년대 ───────────────────────────────────────────────────      │
│   │  학술 네트워크로 확대                                            │
│   │  - NSFNET (미국 과학재단 네트워크)                               │
│   │                                                                 │
│   1990년대 ───────────────────────────────────────────────────      │
│   │  상용 인터넷 시작                                                │
│   │  - WWW 발명 (팀 버너스 리, 1991)                                │
│   │  - 일반인도 인터넷 사용 가능                                     │
│   │                                                                 │
│   2000년대~ ──────────────────────────────────────────────────      │
│      초고속 인터넷, 모바일 인터넷 시대                               │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### ARPANET (알파넷)

| 항목 | 내용 |
|------|------|
| **전체 이름** | Advanced Research Projects Agency Network |
| **시작** | 1969년 12월 |
| **주관** | 미국 국방부 ARPA |
| **초기 노드** | UCLA, 스탠포드, UCSB, 유타대 (4개) |
| **의의** | 인터넷의 시초, 패킷 교환 방식 도입 |

## 요약

### 핵심 비교

```
┌─────────────────────────────────────────────────────────────────────┐
│                          핵심 정리                                   │
│                                                                     │
│   LAN (Local Area Network)                                         │
│   ├── 근거리 통신망                                                 │
│   ├── 건물, 캠퍼스 내부                                             │
│   ├── 고속, 저지연, 낮은 오류율                                     │
│   └── 조직이 직접 구축/관리                                         │
│                                                                     │
│   WAN (Wide Area Network)                                          │
│   ├── 광역 통신망                                                   │
│   ├── 도시, 국가, 전세계                                            │
│   ├── 상대적 저속, 고지연, 높은 오류율                               │
│   └── ISP 서비스 이용                                               │
│                                                                     │
│   MAN (Metropolitan Area Network)                                  │
│   ├── 도시권 통신망                                                 │
│   └── LAN과 WAN 사이 규모                                           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 용어 정리

| 용어 | 설명 |
|------|------|
| **LAN** | Local Area Network, 근거리 통신망 |
| **WAN** | Wide Area Network, 광역 통신망 |
| **MAN** | Metropolitan Area Network, 도시권 통신망 |
| **ISP** | Internet Service Provider, 인터넷 서비스 제공자 |
| **스위치** | 같은 네트워크 내 장비 연결 (L2) |
| **라우터** | 다른 네트워크 간 연결 (L3) |
| **ARPANET** | 인터넷의 시초 (1969년) |

### 핵심 포인트

```
1. LAN = 좁은 범위, 빠른 속도, 직접 관리
2. WAN = 넓은 범위, 느린 속도, ISP 이용
3. 인터넷 = 전세계 WAN들의 연결
4. 가정 공유기 = LAN 구성 + WAN 연결
5. LAN들을 WAN으로 연결하여 인터넷 형성
```
