# 가정에서의 LAN 구성

## 1. LAN과 WAN의 개념

### LAN (Local Area Network)
- **정의**: 집, 사무실, 학교 등 제한된 지역 내에서 컴퓨터와 장치를 연결하는 네트워크
- **특징**:
  - 빠른 전송 속도 (1Gbps ~ 10Gbps)
  - 낮은 지연시간
  - 직접 관리 가능
  - 구축 비용이 상대적으로 저렴

### WAN (Wide Area Network)
- **정의**: 도시, 국가, 대륙 등 넓은 지역을 연결하는 네트워크
- **특징**:
  - LAN에 비해 느린 전송 속도
  - 높은 지연시간
  - ISP가 관리
  - 인터넷이 대표적인 WAN

```
┌─────────────────────────────────────────────────────────────┐
│                         인터넷 (WAN)                          │
└─────────────────────────────────────────────────────────────┘
                              │
                    ┌─────────┴─────────┐
                    │   ISP 네트워크     │
                    └─────────┬─────────┘
                              │
              ┌───────────────┴───────────────┐
              │       인터넷 공유기 (라우터)    │
              └───────────────┬───────────────┘
                              │
        ┌─────────────────────┼─────────────────────┐
        │                     │                     │
   ┌────┴────┐          ┌────┴────┐          ┌────┴────┐
   │   PC    │          │ 스마트폰 │          │  노트북  │
   └─────────┘          └─────────┘          └─────────┘
              ↑────────── LAN (가정 내부) ──────────↑
```

## 2. 인터넷 서비스 제공자 (ISP)

### ISP란?
- **Internet Service Provider**의 약자
- 가정이나 기업에 인터넷 접속 서비스를 제공하는 회사

### 국내 주요 ISP
| ISP | 특징 |
|-----|------|
| KT | 국내 최대 통신사, 광범위한 회선 보유 |
| SK브로드밴드 | SK텔레콤 계열, 결합상품 강점 |
| LG U+ | 인터넷 TV 결합 상품 강점 |
| 케이블사 (CJ헬로 등) | 지역 기반 서비스 |

### ISP가 제공하는 것
1. **공인 IP 주소**: 인터넷에서 유일한 식별자
2. **인터넷 회선**: 물리적 연결
3. **DNS 서버**: 도메인 이름을 IP로 변환
4. **기타 서비스**: 이메일, 웹호스팅 등

## 3. 인터넷 회선의 종류

### 광케이블 (FTTH - Fiber To The Home)
- **특징**: 빛을 이용한 데이터 전송
- **속도**: 100Mbps ~ 10Gbps
- **장점**: 빠른 속도, 안정적, 거리에 따른 속도 저하 없음
- **단점**: 설치 비용이 높음

### DSL (Digital Subscriber Line)
- **특징**: 기존 전화선을 이용
- **속도**: 1Mbps ~ 100Mbps
- **장점**: 기존 인프라 활용, 설치 용이
- **단점**: 거리에 따른 속도 저하

### 케이블 인터넷
- **특징**: 케이블 TV 회선 이용
- **속도**: 10Mbps ~ 1Gbps
- **장점**: 광범위한 커버리지
- **단점**: 이웃과 대역폭 공유

```
속도 비교:

광케이블 (FTTH)  ████████████████████████████████  1Gbps
케이블 인터넷    ████████████████                   500Mbps
VDSL            ████████████                       300Mbps
ADSL            ███                                50Mbps
```

## 4. 인터넷 공유기 (Broadband Router)

### 공유기의 핵심 기능

#### 1) 라우터 기능
- 외부 네트워크(WAN)와 내부 네트워크(LAN) 연결
- 패킷을 목적지로 전달

#### 2) NAT (Network Address Translation)
- **공인 IP 1개로 여러 기기가 인터넷 사용 가능**
- 사설 IP를 공인 IP로 변환

```
내부 네트워크 (사설 IP)              외부 네트워크 (공인 IP)
┌─────────────────────┐             ┌─────────────────────┐
│ PC: 192.168.0.2     │             │                     │
│ 노트북: 192.168.0.3  │ ──[NAT]──→ │ 공인IP: 211.xx.xx.xx │ → 인터넷
│ 폰: 192.168.0.4     │             │                     │
└─────────────────────┘             └─────────────────────┘
```

#### 3) DHCP 서버 기능
- **Dynamic Host Configuration Protocol**
- 자동으로 IP 주소 할당

```
기기 연결 시 DHCP 동작 과정:
1. 기기: "IP 주소 주세요" (DHCP Discover)
2. 공유기: "192.168.0.10 어때요?" (DHCP Offer)
3. 기기: "네, 그걸로 할게요" (DHCP Request)
4. 공유기: "확인, 사용하세요" (DHCP Ack)
```

#### 4) 스위칭 허브 기능
- 여러 유선 기기 연결 (보통 4개 포트)
- MAC 주소 기반 프레임 전달

#### 5) 무선 AP (Access Point) 기능
- Wi-Fi 신호 송출
- 무선 기기 연결 지원

#### 6) 방화벽 기능
- 외부 침입 차단
- 포트 필터링

## 5. 공인 IP vs 사설 IP

### 공인 IP (Public IP)
- **정의**: 인터넷에서 유일하게 식별되는 IP 주소
- **특징**: ISP로부터 할당받음
- **예시**: `211.234.56.78`

### 사설 IP (Private IP)
- **정의**: 내부 네트워크에서만 사용하는 IP 주소
- **특징**: 외부에서 직접 접근 불가

### 사설 IP 대역 (RFC 1918)
| 클래스 | 대역 | 범위 |
|--------|------|------|
| A | 10.0.0.0/8 | 10.0.0.0 ~ 10.255.255.255 |
| B | 172.16.0.0/12 | 172.16.0.0 ~ 172.31.255.255 |
| C | 192.168.0.0/16 | 192.168.0.0 ~ 192.168.255.255 |

```
가정에서 흔히 볼 수 있는 IP 구성:

공유기 게이트웨이: 192.168.0.1 (또는 192.168.1.1)
PC:              192.168.0.2
노트북:           192.168.0.3
스마트폰:         192.168.0.4
스마트TV:         192.168.0.5
```

## 6. 유선 LAN vs 무선 LAN

### 유선 LAN (Ethernet)
| 항목 | 내용 |
|------|------|
| 연결 방식 | 랜 케이블 (UTP, Cat5e/Cat6) |
| 속도 | 100Mbps ~ 10Gbps |
| 장점 | 안정적, 빠른 속도, 보안성 높음 |
| 단점 | 케이블 필요, 이동 제한 |
| 적합한 용도 | 데스크톱 PC, 게이밍, 서버 |

### 무선 LAN (Wi-Fi)
| 항목 | 내용 |
|------|------|
| 연결 방식 | 무선 전파 (2.4GHz, 5GHz, 6GHz) |
| 속도 | 최대 9.6Gbps (Wi-Fi 6) |
| 장점 | 이동 자유, 케이블 불필요 |
| 단점 | 간섭 가능성, 거리에 따른 속도 저하 |
| 적합한 용도 | 스마트폰, 태블릿, 노트북 |

### Wi-Fi 세대별 특징
| 세대 | 규격 | 최대 속도 | 주파수 |
|------|------|-----------|--------|
| Wi-Fi 4 | 802.11n | 600Mbps | 2.4/5GHz |
| Wi-Fi 5 | 802.11ac | 3.5Gbps | 5GHz |
| Wi-Fi 6 | 802.11ax | 9.6Gbps | 2.4/5GHz |
| Wi-Fi 6E | 802.11ax | 9.6Gbps | 2.4/5/6GHz |
| Wi-Fi 7 | 802.11be | 46Gbps | 2.4/5/6GHz |

## 7. 실제 가정 네트워크 구성 예시

### 기본 구성
```
[인터넷] ─── [ISP 모뎀] ─── [공유기] ─┬─ [PC] (유선)
                                     ├─ [노트북] (Wi-Fi)
                                     ├─ [스마트폰] (Wi-Fi)
                                     └─ [스마트TV] (유선/Wi-Fi)
```

### 확장 구성 (넓은 집)
```
                              ┌─ [Wi-Fi 확장기] ─── [원거리 기기]
[인터넷] ─── [공유기] ───────┤
                              └─ [유선 스위치] ─┬─ [PC1]
                                                ├─ [PC2]
                                                └─ [NAS]
```

### IP 주소 설정 확인 방법

**Windows:**
```cmd
ipconfig
```

**결과 예시:**
```
이더넷 어댑터 이더넷:
   IPv4 주소 . . . . . . . . . : 192.168.0.10
   서브넷 마스크 . . . . . . . : 255.255.255.0
   기본 게이트웨이 . . . . . . : 192.168.0.1
```

**Mac/Linux:**
```bash
ifconfig
# 또는
ip addr
```

### 공유기 관리 페이지 접속
1. 브라우저에서 `192.168.0.1` 또는 `192.168.1.1` 입력
2. 관리자 계정으로 로그인 (기본값: admin/admin)
3. 네트워크 설정, 보안 설정 등 관리

## 8. 네트워크 보안 기본 설정

### 필수 보안 조치
1. **공유기 관리자 비밀번호 변경** (기본값 사용 금지)
2. **Wi-Fi 암호화** (WPA3 또는 WPA2-PSK 권장)
3. **SSID 변경** (기본 SSID는 공유기 모델 노출)
4. **펌웨어 업데이트** (보안 취약점 패치)

### Wi-Fi 보안 프로토콜 비교
| 프로토콜 | 보안 수준 | 권장 여부 |
|----------|-----------|-----------|
| WEP | 매우 낮음 | 사용 금지 |
| WPA | 낮음 | 비권장 |
| WPA2-PSK | 양호 | 권장 |
| WPA3 | 높음 | 강력 권장 |

## 핵심 정리

| 구성 요소 | 역할 |
|-----------|------|
| ISP | 인터넷 접속 서비스 제공 |
| 인터넷 회선 | 물리적 연결 (광케이블, DSL 등) |
| 공유기 | NAT, DHCP, 라우팅, 방화벽 기능 수행 |
| 유선 LAN | 안정적이고 빠른 연결 |
| 무선 LAN | 이동성과 편의성 제공 |
| 사설 IP | 내부 기기 식별 (192.168.x.x) |
| 공인 IP | 인터넷상 식별 (ISP 할당) |

{{< hint info >}}
**인터넷 공유기**
영어로 Broadband Router라고 한다. 가정이나 소규모 기업에서 인터넷에 접속할 때 쓰인다. 최근에는 라우터의 기능뿐만 아니라 허브, 스위칭 허브, 방화벽, DHCP 서버, NAT 등 다양한 기능을 통합 제공한다.
{{< /hint >}}
