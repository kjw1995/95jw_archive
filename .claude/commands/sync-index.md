지정한 디렉터리의 _index.md를 실제 파일 목록과 동기화하는 명령어입니다.

## 입력

$ARGUMENTS

축약어 또는 경로 (예: "모던자바", "java/web-programming", "education" 등)
입력이 없으면 **전체** education 하위를 스캔합니다.

## 워크플로우

### 1단계: 대상 디렉터리 결정

`content/education/CLAUDE.md`의 디렉토리 등록부에서 대상 경로를 결정합니다.

### 2단계: 파일 목록 스캔

대상 디렉터리의 `.md` 파일을 모두 조회합니다 (_index.md, CLAUDE.md 제외).
각 파일의 frontmatter에서 `title`과 `weight`를 읽습니다.

### 3단계: 현재 _index.md 읽기

기존 `_index.md`의 테이블/카드 항목을 파싱합니다.

### 4단계: 차이 비교

- **누락된 파일**: _index.md에 없지만 디렉터리에 있는 파일
- **깨진 링크**: _index.md에 있지만 디렉터리에 없는 파일
- **순서 불일치**: weight 순서와 테이블 순서가 다른 경우

### 5단계: 결과 출력

차이를 테이블로 보여줍니다:

```
| 상태 | 파일 | 설명 |
|:-----|:-----|:-----|
| ✅ 정상 | 03-servlet.md | 서블릿 |
| ⚠️ 누락 | 11-new-topic.md | _index.md에 미등록 |
| ❌ 깨진 링크 | old-page | 파일 없음 |
```

### 6단계: 수정 적용

사용자 확인 후 _index.md를 업데이트합니다.
상위 _index.md에도 반영이 필요하면 함께 수정합니다.

### 7단계: 커밋 & 푸시

변경이 있을 경우에만 커밋 후 푸시합니다.
